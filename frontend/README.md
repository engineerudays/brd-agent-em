# Frontend Application

This directory contains the frontend application code.

## Structure

- `src/` - Source code files
- `public/` - Static assets (images, fonts, etc.)
- `components/` - Reusable UI components

## Setup

1. Initialize your frontend framework (React, Vue, Angular, etc.)
2. Place source code in `src/`
3. Store static assets in `public/`
4. Create reusable components in `components/`

## Getting Started

```bash
# Example for React
npm init react-app .
# or for Vue
npm init vue@latest .
```
